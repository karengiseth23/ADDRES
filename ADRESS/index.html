Sistema Integral de Gestión de Adquisiciones

Bienvenido al Sistema Integral de Gestión de Adquisiciones. Este sistema te permite realizar un seguimiento eficiente de los requerimientos de bienes y servicios. A continuación, te proporcionamos una breve guía para utilizar las funciones principales:

  • Crear Registro:

    Completa campos en la sección correspondiente y guarda el registro.

  • Consultar Registros:

    Accede a la sección de consultas y utiliza filtros para revisar detalles.

  • Filtrar Registros:

    Utiliza opciones de filtrado para acceder rápidamente a información específica.

  • Eliminar Registros:

    Accede a la función correspondiente y confirma para mantener la integridad del sistema.

  • Historial de Cambios:

    Visualiza todas las modificaciones realizadas en el historial de cada registro.
